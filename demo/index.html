Quick Start Installation

Get up and running in minutes with our streamlined setup process

Clone the Repository

# Clone the repository git clone https://github.com/spyberpolymath/bdas.git cd bdas

Create Virtual Environment

# Create virtual environment (recommended) python -m venv venv # On Windows: venv\Scripts\activate # On macOS/Linux: source venv/bin/activate

Install Dependencies

# Install dependencies pip install -r requirements.txt # Launch Jupyter Notebook jupyter notebook

Run Automated Pipeline

# Execute complete analysis pipeline python scripts.py # Outputs will be generated in data_output/ directory
Pro Tip: Each notebook includes top cells that set relative data paths. Run those cells first before executing the analysis!
